\section{The Identity Substitution on Effect Environments}
For each type environment $\P$, define the identity substitution $I_{\P}$ as so:

\begin{itemize}
    \item $I_{\nil} = \nil$
    \item $I_{(\P, \a} = (I_{\P}, \a\setto \a)$
\end{itemize}
